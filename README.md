🌐 Сайт-портфолио (Шаблон)<br>
Проект персонального сайта-портфолио с адаптивной версткой.<br>

🛠 Стек технологий:<br>
HTML5 — структура страниц.<br>
CSS3 — стилизация, сетка Bootstrap Grid и кастомные тултипы.<br>
JavaScript (jQuery) — определение IP пользователя, браузера и ОС.<br>
FontAwesome — иконки соцсетей.<br>

📁 Структура проекта:<br>
index.html — главная страница с приветствием.<br>
about.html — страница с подробным резюме, опытом и навыками.<br>
css/ — файлы стилей.<br>
js/ — скрипты.<br>
img/ — графические ресурсы.<br>

📥 Как использовать:<br>
Скачайте проект (кнопка Code -> Download ZIP).<br>
Откройте index.html в любом современном браузере.<br>
Код можно использовать как основу для создания собственного сайта-визитки.<br>
Проект создан для демонстрации навыков верстки и работы с фронтенд-технологиями.
